Today, the [ImageJ team](Contributors ) has made a new release of [ImageJ2](ImageJ2 ): version 2.0.0-rc-30. This release includes bug fixes and usability improvements, especially for [scripting](Scripting ).

In addition, the [Fiji team](Contributors ) has updated several components of the [Fiji](Fiji ) distribution.
