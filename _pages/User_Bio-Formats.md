This account is used to manage the [http://sites.imagej.net/Bio-Formats/ Bio-Formats update site]. See also {{ListOfUpdateSites}}.

To activate it in your Fiji setup:

* [[Update_Sites|{{bc|Help|Update...}}]]
* Click ''Manage update sites''
* Check ''Bio-Formats''
* Click ''Close''

Some files should appear on the list with ''Status/Action'' of ''Update it''.

* Click ''Apply changes''
* Restart Fiji
* {{bc|Help|About Plugins|Bio-Formats Plugins...}}

The dialog should now show a very recent build date (within the last few days).
