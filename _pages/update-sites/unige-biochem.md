This page documents the [UNIGE-Biochem update site](https://sites.imagej.net/UNIGE-Biochem/), which provides plugins developed and maintained at the [University of Geneva's Biochemistry department](https://www.unige.ch/sciences/biochimie/). Some of the plugins (ABBA, Warpy, bdv-playground) have initially been developped at the [BIOP](https://www.epfl.ch/research/facilities/ptbiop/).

To activate it:

-   [{% include bc path='Help|Update...' %}](/update-sites)
-   Click {% include button label="Manage update sites" %}
-   Check the *UNIGE-Biochem* box
-   Click {% include button label="Close" %}

Some files should appear on the list with *Status/Action* of *Install it*.

-   Click *Apply changes*
-   Restart ImageJ

Your ImageJ should now include additional plugins:

* [ABBA](/plugins/bdv/abba/abba)
* [BigDataViewer-Playground](/plugins/bdv/playground/bdv-playground)
* [Warpy](/plugins/bdv/warpy/warpy)
* [Kheops](https://github.com/BIOP/ijp-kheops)

Since June 2026, this update site contains the most recent versions of these plugins, some of which have been migrated from the PTBIOP update site.

See also [list of update sites](/list-of-update-sites).
