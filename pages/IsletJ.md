{% include info-box content=includecontent %} IsletJ is a plugin for
analysis of microscopy images of Langerhans islets in
[ImageJ](ImageJ "wikilink"). It provides free, open source tool for
analysis of islets diameter and number and for estimation of sample
purity (ratio of endocrine/exocrine tissue area).

## Installation

IsletJ was designed to work with plain [ImageJ
1.x](ImageJ_1.x "wikilink"). It requires Java-8.

The simplest way to install the plugin is using the Fiji installer\>

  - Download current version of Fiji from
    [here](Fiji/Downloads "wikilink")
  - Launch Fiji and run Updater {% include bc content='Help |
    Update...'%}
  - In {% include bc content='ImageJ Updater'%} select {% include bc
    content='Manage update sites'%}
  - In {% include bc content='Manage update sites'%} select {% include
    bc content='IsletJ'%} and then {% include bc content='Close'%}
  - In {% include bc content='ImageJ Updater'%} select {% include bc
    content='Apply changes '%}
  - Restart Fiji
